Arbalest Goose 4 Avionics Systems & Flight Computer Software

This project involves the design and development of a flight computer for our 2-Stage High Power Rocket, set to launch at Launch Canada 2024.

Flight Computer

Our flight computer setup includes two unique PCBs for different rocket stages:

  • Stage 1 ("BLOB"): Manages the first stage operations, ensuring smooth launch and ascent.
  • Stage 2 (Prototyping Board): Controls the second stage and RunCam devices for video recording.

This dual-PCB setup ensures specialized and reliable performance throughout the flight.

Components

  • Microcontroller: Teensy 4.1
  • Sensors: Adafruit BMP280 (Altimeter), Adafruit BNO055 (IMU), SparkFun GNSS Receiver Breakout - MAX-M10S (Qwiic) (GPS)
  • SD Breakout: Adafruit SPI Flash SD Card - XTSD 512 MB
  • Communication: Adafruit LoRa RFM9X
  • COTS Flight Computer: Featherweight Blue Raven
  • COTS GPS: Featherweight GPS Tracker

Priority Search Tree (PST)

This implementation includes methods for building the PST using bottom-up heap construction, performing heapify operations, and printing the tree for visualization purposes.

How does it work?

A Priority Search Tree organizes points in a balanced binary tree structure, where each node represents a range along one dimension. This allows for efficient range queries and point insertion and deletion operations.

Project Details

The PrioritySearchTree.java project implements a Priority Search Tree (PST) in Java. It provides functionality to construct a PST from a set of points and visualize the resulting tree structure. The implementation includes methods for building the PST using bottom-up heap construction, performing heapify operations, and printing the tree for visualization purposes.

Usage

To use the PrioritySearchTree.java implementation, follow these steps:

  • Compile the PrioritySearchTree.java file.
  • Create an array of points (external nodes) to represent the input data.
  • Initialize a PrioritySearchTree object with the array of points.
  • Call the buildPrioritySearchTree() method to construct the PST.
  • Call the printTree() method to visualize the resulting tree structure.

Example


public class Main {
    public static void main(String[] args) {
        Point[] points = { 
            new Point("p1", -8, 3), new Point("p2", -7, 1), 
            new Point("p3", -1, 6), new Point("p4", 2, 4),
            new Point("p5", 4, 8), new Point("p6", 5, 9), 
            new Point("p7", 7, 1), new Point("p8", 9, 7) 
        };

        PrioritySearchTree.testPrioritySearchTree("Example", points);
    }
}
